CLASE OCHO: La estructura de una pagina web

  • <!DOCTYPE html>: Esta linea le dice al navegador que el documento es un archivo html.
  • <html>: Aqui es donde se colocan las etiquetas que no se ven en la pagina, como el titulo de la pagina y los enlaces a otros archivos.
  • <body>: Esta es la parte de la pagina que se muestra en el navegador. Aqui es donde colocamos todo lo que queremos que la gente vea en nuestra pagina web.

CLASE NUEVE: Enlazar diferentes páginas

Para hacer que las personas puedan ir de una pagina a otra usaremos la etiqueta code <a>. Dentro de esta etiqueta, escribimos el texto que queremos que se vea en el enlace y luego usamos el atributo href para decirle al navegador a donde debe llevar a la persona cuando hagan clic en el enlace. Clase 10: Organizando con la etiqueta div

La etiqueta div es como una caja donde puedes pones cualquier cosa que quieras. sirve para organizar y estructurar tu pagina web.Puedes poner varios elememtos dentro de un div para agruparlos y darles estilo

¡Esto es un div!

En este div puedes poner texto imagenes, videos o cualquier otro elemento que desees.

Colocar el video

Para poner un video en nuestra pagina web, usaremos la etiqueta <video>.Dentro de esta etiqueta la ruta del archivo de video utilizando el atributo src tambien agregamos el atributo controls para que aparezcan los controles de reproduccion del video y la persona puede reproducirlo, para usarlo y ajustar el volumen
